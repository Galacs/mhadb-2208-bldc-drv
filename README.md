# MHADB BLDC DRIVER

STM32F1 based 2208 size gimbal bldc foc driver controller board

## IC List

- STM32F103C8T (Main MCU)
- DRV8311 (BLDC driver)
- AS5600 (magnetic encoder)
- SN65HVD (CAN transceiver)
